# Gallery Viewer
originally created to organize my album gallery. It also supports nested albums.
example gallery available (./example)

# Resource Schema
index.json
```json
{
  "title": "Gallery",
  "albums": [
    "album1",
    "album2"
  ],
  "images": [
    "image1.jpg",
    "image2.jpg",
    "image3.jpg"
  ]
}
```
